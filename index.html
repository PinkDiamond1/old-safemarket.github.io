SafeMarket

SafeMarket is (1) trustless, (2) decentralized, and (e) pseudonymous marketplace.

  1. Trustless: None of your private keys leave your computer.
  2. Decentralized: Data is written to the blockchain, meaning every bitcoin node holds a signed, provable copy.
  3. Pseudonymous: Your transactions can be linked together, but never linked to a real-world identity assuming you practice good bitcoin hygeine.

This software is still in testing and will likely have breaking changes, but I encourage you to try out the beta. Subscribe to /r/SafeMarket for updates.

Getting started

  1. Download the add-on
  2. In Firefox or TorBrowser, head to the Add-Ons menu (its located under the burger menu right of the URL bar).
  3. Click the gear icon, then "Install Add-on From File" and select the safemarket.xpi file you previously downloaded.
  4. Head to any page and right click anywhere on the page. Then select "Open SafeMarket".

How

SafeMarket allows vendors to create special types of bitcoin transactions that add specific information to the blockchain. This information contains things like the vendor's product catalog and pgp key. By supplying the vendor's xpubkey, anyone can download this information directly from the blockchain.

Buyers can select the products they wish to buy and the quantities they wish to purchase. SafeMarket will generate a cryptographically secure receipt containing the order and shipping information. SafeMarket does not provide a method for receipt exchange, it is assumed buyers and sellers will use tools such as email, IM, and bitmessage to exchange receipts.

To anyone listening, a receipt looks like gibberish. But the buyer and seller can indepently decrypt the receipt and validate the status of an order.

Security

IP Obfuscation

If you are using TorBrowser, all outgoing requests will be made through the Tor network. This means your IP address will not be revealed.

XSS Protection

SafeMarket uses Angular for text sanitation. We only allow requests to the two domains necessary for SafeMarket to function.

You can view those domains.

Browser Fingerprinting

You may have heard that Add-Ons like SafeMarket can be used to "fingerprint" you. This is true of some add-ons, but it is not true of SafeMarket. Some add-ons, like ad-blockers, need to inject various scripts and stylesheets onto the pages you visit in order to function. SafeMarket does not inject any resources, and therefore can not be used for fingerprinting.

You can verify that SafeMarket does not use any of the SDKs necessary to fingerprint you.
